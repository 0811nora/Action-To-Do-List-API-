## 專案由來 
主要是透過學習六角學院線上課程中的實作作品練習，實作出的待辦清單功能的網頁，
在此項任務中，課程只有提供介面的畫面，學員需要透過畫面自行做出網頁的功能及呈現。

## 專案介紹 
此專案為『待辦清單』的功能網頁，

## 功能特色  
- **新增任務**：快速新增待辦事項。  
- **編輯任務**：可隨時修改已新增的事項內容。  
- **標記完成**：勾選完成的事項，方便追蹤進度。  
- **刪除任務**：刪除不需要的事項，保持清單整潔。


## 實際使用到的技術：
- HTML
- CSS  
- JavaScript
